# Exporting

## Single Export

Export your current mockup as a high-resolution image.

1. Open the **Export** panel in the sidebar
2. Choose format: **PNG** or **WebP**
3. Click **Export**
4. Choose where to save the file

Exports are rendered at **2x resolution** for crisp retina displays.

## Batch Export (Pro)

Export your mockup in multiple sizes at once — perfect for app store submissions and social media.

### Export Packs

Pre-configured size packs for common platforms:

| Pack | Sizes | Description |
|------|-------|-------------|
| **App Store** | 4 sizes | iPhone 11 Pro Max, XS Max, Pro, 8 Plus dimensions |
| **Play Store** | 4 sizes | Galaxy sizes + feature graphic (1024x500) |
| **Social Media** | 6 sizes | Instagram (square + story), Twitter/X, Facebook, LinkedIn, OG image |
| **Mac App Store** | 2 sizes | MacBook Pro 15" and Air 13" dimensions |
| **iPad App Store** | 2 sizes | iPad Pro 13" and iPad dimensions |

### How Batch Export Works

1. Select an export pack or choose custom sizes
2. Click **Export All**
3. Choose an output folder
4. Frameup renders each size with the appropriate device frame and saves all files

Each export in a batch is automatically named with the target dimensions.

## Format Options

| Format | Best For |
