# Getting Started

## Installation

**From source** (recommended during beta):

```bash
git clone https://github.com/amirmun99/FrameUp-Free.git
cd FrameUp-Free
npm install
npm run dev
```

The app opens in a desktop window. No account or environment setup required.

**Requirements:** Node.js 20+, macOS or Windows

## First Launch

When you open Frameup for the first time, you'll see a short welcome tour. After completing it, you'll land on the **Home** screen.

### Quick Start

1. Click a **template** or go to **Capture** in the sidebar
2. Choose a capture source (URL, upload, or another method)
3. Your screenshot loads into the **Editor** with a device frame
4. Customize the background, device, and overlays
5. Click **Export** to save your mockup

## Interface Overview

Frameup has four main areas:

| Area | Description |
|------|-------------|
| **Sidebar (left)** | Navigation rail — Home, Capture, Editor, Settings |
| **Canvas (center)** | Live preview of your mockup with zoom and pan |
| **Panels (right)** | Source, Device, Background, Overlay, and Export controls |
| **Title bar (top)** | Window controls and status |

The canvas supports:
- **Scroll to zoom** (or Cmd/Ctrl + scroll)
- **Click and drag** to pan
- **Click overlays** to select and reposition them
