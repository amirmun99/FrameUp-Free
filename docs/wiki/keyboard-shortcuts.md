# Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + S` | Save project |
| `Cmd/Ctrl + 0` | Fit canvas to viewport |
| `Cmd/Ctrl + 1` | Zoom to 100% |
| `Cmd/Ctrl + Scroll` | Zoom in/out |
| `Delete` / `Backspace` | Delete selected overlay |
| `Escape` | Deselect overlay |

On **Windows**, use `Ctrl` in place of `Cmd`.
